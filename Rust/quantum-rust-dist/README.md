## Installation

```bash
./install.sh
```

## Quick Start

```rust
use quantum::prelude::*;

fn main() {
    let quantum_vec = quantum_vec![1, 2, 3, 4];
    let result = quantum_fft!(quantum_vec);
    println!("Quantum result: {:?}", result);
}
```
