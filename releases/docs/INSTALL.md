# Quantum Rust 安装指南

## 系统要求

- Linux, macOS, 或 Windows
- 4GB+ RAM
- 2GB+ 可用存储空间

## 安装方法

### 自动安装

```bash
curl -sSf https://quantum-rust.org/install.sh | sh
```

### 手动安装

1. 下载发行版
2. 解压到目标目录
3. 运行 `./install.sh`

## 验证安装

```bash
rustc --version
cargo --version
```

## 卸载

```bash
~/.quantum-rust/uninstall.sh
```
